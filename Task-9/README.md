## Task-9: Complex Responsive Layout using Grid and Flexbox (E-Commerce Landing Page)

## Features implemented:

## Step-1: Page structure using semantic HTML
    ->Used semantic elements such as <header>, <aside>, <main>, and <footer>
    ->Structured the page into header, sidebar, main content, and footer sections

## Step-2: Main layout using CSS Grid
    ->Applied CSS Grid to the parent container
    ->Defined columns for sidebar and main content

## Step-3: Inner layout using Flexbox
    ->Used Flexbox inside the main content area
    ->Aligned product cards horizontally
    ->Controlled spacing and wrapping using gap and flex-wrap

## Step-4: Sticky header and sidebar
    ->Implemented a sticky header using position: sticky
    ->Header remains visible at the top while scrolling

## Step-5: Overlapping elements using position and z-index
    ->Product cards are positioned relatively
    ->Discount badges are positioned absolutely
    ->Used z-index to layer the badge above product images
    ->Demonstrated stacking and overlapping control in CSS

## Step-6: Dynamic reordering based on viewport size
    ->Used media queries to handle responsive behavior
    ->Changed grid structure to a single-column layout on smaller screens
    ->Reordered sidebar and main content using the order property
