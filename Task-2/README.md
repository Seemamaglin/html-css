## Task-2:CSS card components with hover effects

## Step-1:
    ->Create a container <section> for all cards
    ->Create multiple <div> inside containers
    ->Each card contained a tourist place image,title,and description

## Step-2:Flexbox layout
    ->Apply display:flex on container
    ->Used justify content and gap to place cards neatly

## Step-3:Card styling
    ->Add border,padding and box-shadow 

## Step-4:Hover effects
    ->Use :hover on .card to make it interactive.
    ->Apply transform: scale(1.05) to slightly grow the card.
    ->Increase shadow and optionally change background color.
    ->Use transition for smooth animations.
